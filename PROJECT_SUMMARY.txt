================================================================================
USAGE WORKFLOW
================================================================================

Setup (one time):
1. Install dependencies: pip install -r requirements.txt
2. Configure API keys: cp .env.example .env && edit .env
3. Add documents to data/documents/

Run (every time):
1. python main.py
2. System loads config and initializes components
3. Documents are loaded and indexed (first run) or index is loaded (subsequent)
4. Interactive Q&A session begins
5. Ask questions about NIS-2 compliance
6. System retrieves relevant documents and generates answers
7. Sources are cited for verification
